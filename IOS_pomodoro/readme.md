# Ios_Pomodoro(스탑 워치)

## 사용한 기술

* Swift 5, Xcode 12.5.1
* GCD, UIView Animation

## 수행 역할

* GCD를 이용해 매 초 시간을 업데이트
* UIView Animation 함수를 이용한 부드러운 전환
